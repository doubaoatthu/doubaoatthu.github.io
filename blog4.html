Fixed Cuda Bug: cannot open shared object file

Posted on August 29, 2016 | keras, cuda,

I got the following error on cuda

cuda_ndarray.cu: libcublas.so.7.0:cannot open shared object file

And I solved it using the following line

sudo ldconfig $LD_LIBRARY_PATH

This is all I know so far. If you have any questions, please feel free to email me.
