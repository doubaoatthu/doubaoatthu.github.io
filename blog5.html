Solved: How to download thunder files in linux.

Run the following code in Terminal to get the decrypted url address

echo thunderurl|base64 -d

Solved: How to format a USB disk

First, run the following command to check which device boot is the usb disk

sudo fdisk -l

Then, run the following to off-load the usb disk.

umount path/to/dev/sdc4

After that, you can safely format the usb disk using the following command.

sudo mkfs.vfat path/to/dev/sdc4

Solved: How to write an ISO file into your usb disk on Linux

First, refering the last problem, check the location of usb disk and off-load it.

sudo fdisk -l
umount path/to/dev/sdc4

Then, use the following command.

sudo dd if=path/to/iso/file of=path/to/usb/disk/dev/sdc4

It will take a few munites to finish.

Solved: How to make your desktop ssh achievable.

You need to check if ssh server is installed properly. Whether the user is permitted. And other iptable rules that might forbid ssh input/output.

ps -ef | grep sshd
sudo iptables -L
ls -lrt/etc/ssh

To install ssh server, type:

sudo apt-get install openssh-server

If it has not started automantically, start it by:

sudo /etc/init.d/ssh start

The default port number is 22, configure other ports by modifying the configure file located at /etc/ssh/sshd_config, and then restart the ssh server:

sudo /etc/init.d/ssh stop
sudo /etc/init.d/ssh start
